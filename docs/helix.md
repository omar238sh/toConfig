# Helix Editor

The `toconfig::helix` module generates Helix editor configuration (`~/.config/helix/config.toml`) from typed Rust structs. Nodes implement the `HelixConfig` trait and are collected in a `HelixConfigTree`. The trait is **distinct** from `Config` and `HyprlandConfig` so nodes cannot be accidentally mixed.

## HelixConfigTree

```rust
use toconfig::helix::{HelixConfig, HelixConfigTree};

let tree = HelixConfigTree::new()
    .with_comment("config.toml — generated by toconfig")
    .push(/* any HelixConfig node */);

println!("{}", tree.generate());
```

## Editor section

The `EditorSection` maps to the `[editor]` block in `config.toml`.

```rust
use toconfig::helix::editor::{
    EditorSection, LineNumber, CursorKind, CursorShapeSection,
    FilePickerSection, LspSection, SearchSection, StatuslineSection,
    SoftWrapSection, IndentGuidesSection, Bufferline, StatuslineElement,
};

// Core editor settings
EditorSection::new()
    .line_number("relative")        // "absolute" | "relative"
    .mouse(false)
    .scrolloff(5)
    .scroll_lines(3)
    .shell(&["bash", "-c"])
    .rulers(&[80, 120])
    .auto_save(true)
    .idle_timeout(400)
    .completion_trigger_len(2)
    .bufferline(Bufferline::Always) // Always | Never | Multiple
    .color_modes(true);

// Cursor shapes per mode
CursorShapeSection::new()
    .normal(CursorKind::Block)
    .insert(CursorKind::Bar)
    .select(CursorKind::Underline);

// File picker
FilePickerSection::new()
    .hidden(false)     // show hidden files
    .follow_symlinks(true);

// LSP
LspSection::new()
    .enable(true)
    .display_messages(true)
    .display_inlay_hints(true);

// Search
SearchSection::new()
    .smart_case(true)
    .wrap_around(true);

// Status line
StatuslineSection::new()
    .left(&[StatuslineElement::Mode, StatuslineElement::Spinner])
    .center(&[StatuslineElement::FileName])
    .right(&[StatuslineElement::Diagnostics, StatuslineElement::Position]);

// Soft wrap
SoftWrapSection::new()
    .enable(true)
    .wrap_at_text_width(true);

// Indent guides
IndentGuidesSection::new()
    .render(true)
    .character("│");
```

## Keys section

The `KeysSection` maps to the `[keys.normal]`, `[keys.insert]`, or `[keys.select]` blocks.

```rust
use toconfig::helix::keys::{KeysSection, KeyMode, KeyAction, KeyBinding};

// Normal mode bindings
KeysSection::new(KeyMode::Normal)
    .cmd("C-s", ":write")
    .cmd("C-q", ":quit")
    .cmd("space-f", ":open ~/.config/helix/config.toml");

// Insert mode
KeysSection::new(KeyMode::Insert)
    .cmd("j-k", "normal_mode");   // jk → escape to normal mode
```

For nested key sequences (e.g. `space` → submenu), use `KeyBinding::map(key, subtable)`:

```rust
KeysSection::new(KeyMode::Normal)
    .bind(KeyBinding::map("space", vec![
        ("f".into(), KeyAction::Cmd(":file-picker".into())),
        ("b".into(), KeyAction::Cmd(":buffer-picker".into())),
    ]));
```

## Generic TOML sections

For any TOML block not covered by the typed sections, use `TomlSection` or `TomlEntry`:

```rust
use toconfig::helix::section::{TomlSection, TomlEntry, RawToml};

let section = TomlSection::new("editor.whitespace.render")
    .entry(TomlEntry::new("space", "\"dot\""))
    .entry(TomlEntry::new("tab",   "\"arrow\""));

// Raw TOML escape hatch
let raw = RawToml::new("[language]\nname = \"rust\"");
```

## End-to-end example

```rust
use toconfig::helix::{HelixConfig, HelixConfigTree, ThemeSetting};
use toconfig::helix::editor::{
    EditorSection, LineNumber, CursorShapeSection, CursorKind,
    LspSection, IndentGuidesSection,
};
use toconfig::helix::keys::{KeysSection, KeyMode};
use toconfig::output::ConfigOutput;

fn main() -> std::io::Result<()> {
    let tree = HelixConfigTree::new()
        .with_comment("config.toml — generated by toconfig")
        .push(ThemeSetting::new("catppuccin_mocha"))
        .push(
            EditorSection::new()
                .line_number("relative")
                .mouse(false)
                .scrolloff(5)
                .auto_save(true),
        )
        .push(
            CursorShapeSection::new()
                .normal(CursorKind::Block)
                .insert(CursorKind::Bar),
        )
        .push(LspSection::new().enable(true).display_inlay_hints(true))
        .push(IndentGuidesSection::new().render(true))
        .push(
            KeysSection::new(KeyMode::Normal)
                .cmd("C-s", ":write")
                .cmd("C-q", ":quit"),
        )
        .push(
            KeysSection::new(KeyMode::Insert)
                .cmd("j-k", "normal_mode"),
        );

    let path = format!("{}/.config/helix/config.toml", std::env::var("HOME").unwrap());
    ConfigOutput::at_path(&path).write(&tree)?;
    Ok(())
}
```
