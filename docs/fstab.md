# /etc/fstab

The `toconfig::fstab` module generates `/etc/fstab` files from typed Rust structs. Each mount point is represented as a `FstabEntry`; entries are collected in a `Fstab`. Typed option builders for common, ext4, btrfs, tmpfs, and NFS filesystems live in the `options` sub-module.

## Fstab and FstabEntry

```rust
use toconfig::fstab::{Fstab, FstabEntry};

let fstab = Fstab::new()
    .with_header("# /etc/fstab — generated by toconfig")
    .add(
        FstabEntry::new("UUID=abc-123", "/", "ext4")
            .options_str("defaults,noatime")
            .dump(0)
            .pass(1),
    );

println!("{}", fstab.generate());
```

### `FstabEntry` fields

| Method | Column | Default |
|--------|--------|---------|
| `FstabEntry::new(device, mountpoint, fstype)` | 1–3 | required |
| `.options_str(s)` | options (col 4) | `"defaults"` |
| `.dump(n)` | dump (col 5) | `0` |
| `.pass(n)` | pass (col 6) | `0` |

## Mount options

The `options` sub-module provides typed builders that produce the options string. Multiple builders can be chained — their results are joined with commas.

```rust
use toconfig::fstab::options::{CommonOptions, BtrfsOptions, ExtOptions, TmpfsOptions, NfsOptions};

// Common options (work with any filesystem)
CommonOptions::new()
    .noatime()
    .noexec()
    .nosuid()
    .nodev()
    .ro()           // read-only
    .sync()
    .build()
// → "noatime,noexec,nosuid,nodev"

// Btrfs-specific
BtrfsOptions::new()
    .subvol("@root")
    .compress("zstd:3")
    .ssd()
    .discard_async()
    .space_cache("v2")
    .build()
// → "subvol=@root,compress=zstd:3,ssd,discard=async,space_cache=v2"

// ext4-specific
ExtOptions::new()
    .journal_checksums()
    .data("ordered")
    .barrier(1)
    .build()

// tmpfs-specific
TmpfsOptions::new()
    .size("4G")
    .mode("1777")
    .uid("0")
    .gid("0")
    .build()
// → "size=4G,mode=1777"

// NFS-specific
NfsOptions::new()
    .vers(4)
    .rsize(1048576)
    .wsize(1048576)
    .hard()
    .timeo(600)
    .retrans(2)
    .build()
// → "vers=4,rsize=1048576,wsize=1048576,hard,timeo=600,retrans=2"
```

Call `.options(builder.build())` on `FstabEntry` to append a built options string:

```rust
FstabEntry::new("UUID=abc-123", "/", "btrfs")
    .options(BtrfsOptions::new().subvol("@root").compress("zstd:3").ssd().build())
    .options(CommonOptions::new().noatime().build())
    .pass(0)
```

## Convenience constructors

```rust
// tmpfs shortcut
FstabEntry::tmpfs("/tmp")
    .options(TmpfsOptions::new().size("4G").mode("1777").build())
    .options(CommonOptions::new().noexec().nosuid().nodev().build())

// swap shortcut
FstabEntry::swap("UUID=SWAP-999")
// → UUID=SWAP-999  none  swap  sw  0  0
```

## End-to-end example

```rust
use toconfig::fstab::{Fstab, FstabEntry};
use toconfig::fstab::options::{BtrfsOptions, CommonOptions, TmpfsOptions};
use toconfig::output::ConfigOutput;

fn main() -> std::io::Result<()> {
    let fstab = Fstab::new()
        .with_header("# /etc/fstab — generated by toconfig")
        // Root partition (btrfs, @root subvolume)
        .add(
            FstabEntry::new("UUID=abc-123", "/", "btrfs")
                .options(
                    BtrfsOptions::new()
                        .subvol("@root")
                        .compress("zstd:3")
                        .ssd()
                        .discard_async()
                        .space_cache("v2")
                        .build(),
                )
                .options(CommonOptions::new().noatime().build())
                .pass(0),
        )
        // Home subvolume
        .add(
            FstabEntry::new("UUID=abc-123", "/home", "btrfs")
                .options(BtrfsOptions::new().subvol("@home").compress("zstd:3").ssd().build())
                .options(CommonOptions::new().noatime().build())
                .pass(0),
        )
        // EFI partition
        .add(
            FstabEntry::new("UUID=EFI-0000", "/boot/efi", "vfat")
                .options_str("umask=0077")
                .dump(0)
                .pass(2),
        )
        // tmpfs for /tmp
        .add(
            FstabEntry::tmpfs("/tmp")
                .options(TmpfsOptions::new().size("4G").mode("1777").build())
                .options(CommonOptions::new().noexec().nosuid().nodev().build()),
        )
        // Swap
        .add(FstabEntry::swap("UUID=SWAP-999"));

    ConfigOutput::at_path("/etc/fstab").write(&fstab)?;
    Ok(())
}
```
